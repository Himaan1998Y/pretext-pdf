pretext-pdf

The PDF library AI agents speak natively. A PdfDocument is plain JSON — LLMs emit it in one shot, no codegen, no headless browser.

Drop into Claude / Cursor / Windsurf via MCP

{
  "mcpServers": {
    "pretext-pdf": {
      "command": "npx",
      "args": ["-y", "pretext-pdf-mcp"]
    }
  }
}

Tools exposed: generate_pdf, generate_invoice, generate_report, generate_from_markdown, list_element_types.

Install

npm install pretext-pdf
import { render } from 'pretext-pdf'

const pdf = await render({
  type: 'document',
  children: [
    { type: 'text', content: 'Hello, pretext-pdf!', fontSize: 24, bold: true }
  ]
})

// pdf is a Uint8Array — write to disk, stream it, return it from an API

Features

📄

Rich text & typography

Kerning, hyphenation, RTL/CJK, inline bold/italic/code spans.

📊

Tables & charts

Vega-Lite charts, data tables with column sizing and borders.

🖼️

Images & SVG

Embed PNG/JPEG from URL or base64, SVG inline or from file.

🔲

QR codes & barcodes

Generate QR codes and Code128 barcodes from any string.

📝

Markdown input

Pass raw Markdown — it's converted to a typed document tree automatically.

🧾

Invoice & report templates

One-call helpers for common business documents including GST invoices.
